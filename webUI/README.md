# Basic Handwritten Notes Converter WebUI

## Installation
1. Install Python 3.10+ and Git.
2. Clone: `git clone https://github.com/yourusername/handwritten-notes-converter.git`
3. Navigate: `cd handwritten-notes-converter`
4. Install deps: `pip install -r requirements.txt`

## Running
- Linux/macOS: `./webui.sh`
- Windows: Double-click `webui.bat`
- Access: Browser at `http://127.0.0.1:7860`.
- Usage: Upload image, click Convert for B&W version.
